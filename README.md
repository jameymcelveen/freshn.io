**Freshn** is a private CLI tool designed to automate the setup and maintenance of developer workstations. It uses a declarative `freshn.toml` file to ensure your environment is always in sync.

## 🚀 Quick Start
```bash
make install
make fresh
```

## 🛠 Features
- **Declarative TOML:** Simple, human-readable config.
- **Homebrew Integration:** Syncs packages and casks automatically.
- **Dotfile Symlinking:** Keep your `.zshrc`, `.gitconfig`, and aliases in GitHub.
- **Idempotent:** Run it 1,000 times; it only changes what is necessary.
